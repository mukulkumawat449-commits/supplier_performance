# 📊 Supplier Performance Prediction System

A Machine Learning-powered Flask web application that predicts supplier performance as **Good Performer** or **Poor Performer** based on key supplier metrics. The project helps businesses evaluate supplier reliability, improve procurement decisions, and optimize supply chain management through data-driven insights.

## 🚀 Features

* 🤖 Supplier Performance Prediction
* 📈 Multiple ML Model Comparison
* 📊 Interactive Data Visualizations
* 🎯 Performance Evaluation Metrics
* 🔍 Feature Importance Analysis
* 🌐 User-Friendly Flask Web Interface
* 📡 REST API for Model Metrics

## 🛠️ Tech Stack

* Python
* Flask
* Scikit-learn
* Pandas
* NumPy
* Matplotlib
* Seaborn
* HTML, CSS, Bootstrap

## 📂 Machine Learning Models

* Logistic Regression
* Decision Tree Classifier
* Random Forest Classifier
* Gradient Boosting Classifier

## 📊 Business Impact

This project enables organizations to identify high-performing suppliers, reduce supply chain risks, improve vendor selection strategies, and make informed procurement decisions using predictive analytics.

## 🎯 Objective

To build an end-to-end machine learning solution that analyzes supplier data, predicts future performance, and presents actionable insights through an interactive web dashboard.
